## 🎯 项目目标

打造一个功能完整、界面精美的暗黑4模拟器，支持伤害计算、装备模拟、技能模拟、巅峰盘规划等核心功能。

## 🔧 技术路线

### 前端架构
- **新版模块**：React 18 + TypeScript + Vite
- **状态管理**：Zustand
- **样式框架**：Tailwind CSS 3
- **旧版保留**：`modules/simulator/` 使用原生JS

### UI/UX
- 完善暗黑风格设计
- 增强动画效果
- 优化响应式布局

### 数据管理
- 扩展游戏数据
- 支持数据版本管理
- 提供数据API
