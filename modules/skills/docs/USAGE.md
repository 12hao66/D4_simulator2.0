# 技能模拟器 使用指南

## 🎮 基本操作

### 1. 选择职业

1. 点击顶部的职业选择器下拉框
2. 选择想要模拟的职业（野蛮人、德鲁伊、死灵法师、圣骑士、游侠、法师、灵巫、术士）
3. 技能树会切换到对应职业的技能配置

### 2. 选择技能

1. 点击技能树中的技能节点
2. 技能会高亮显示（金色边框+发光效果）
3. 右侧面板会显示已选技能列表和属性加成
4. 技能等级会自动提升（最多提升到最大等级）

### 3. 技能等级提升

1. 多次点击同一技能可提升等级
2. 每个技能有最大等级限制
3. 等级提升会增加技能效果和伤害
4. 技能面板会显示当前等级和加成效果

### 4. 取消技能

1. 再次点击已选技能可取消选择
2. 如果技能有后续技能依赖，会提示确认
3. 取消后技能点数会返还

### 5. 保存Build

1. 点击"保存Build"按钮
2. 输入Build名称
3. 点击确定保存到本地存储

### 6. 加载Build

1. 点击"加载Build"按钮
2. 从列表中选择要加载的Build
3. 点击确定加载配置

### 7. 导出配置

1. 点击"导出配置"按钮
2. JSON配置自动复制到剪贴板
3. 可用于分享或备份

### 8. 编辑模式

1. 点击"编辑模式"按钮进入编辑状态
2. 点击并拖拽技能节点可调整位置
3. 使用编辑器面板修改节点属性（名称、图标、类型等）
4. 添加或删除技能节点
5. 再次点击"编辑模式"按钮退出编辑状态

## 📊 界面说明

### 主区域（技能树）
- 职业技能树可视化展示（Canvas渲染）
- 技能节点按分类排列：核心、基础、防御、攻击、终极、被动
- 技能节点颜色区分：
  - 主动技能：蓝色
  - 被动技能：绿色
  - 终极技能：橙色
  - 核心技能：金色
- 连接线：已解锁技能显示实线，未解锁显示虚线
- 支持鼠标滚轮缩放（0.5x - 2x）
- 支持拖拽移动视图
- 编辑模式下显示虚拟网格线

### 左侧面板
- 技能分类列表
- 显示各分类的技能数量
- 支持展开/折叠分类

### 右侧面板
- **技能详情**：选中技能的详细信息
- **已选技能列表**：当前Build的技能配置
- **属性统计**：力量、敏捷、智力、意志等属性加成
- **技能点数**：总数/已消耗
- **编辑器面板**：编辑模式下显示节点属性编辑表单

### 顶部栏
- 职业选择器（8种职业）
- 缩放控制按钮
- 重置视图按钮
- 编辑模式切换按钮
- Build管理按钮（保存/加载/导出）

## 🔗 技能解锁规则

- **核心技能**：初始解锁，无需前置技能
- **基础技能**：无前置需求，可直接选择
- **防御/攻击技能**：可能需要核心技能达到一定等级
- **终极技能**：需要多个前置技能解锁
- **被动技能**：部分有前置技能需求

## 💡 提示

### 技能选择技巧
- 先选择核心技能作为基础
- 根据Build方向选择防御或攻击技能
- 终极技能通常需要较多前置技能
- 注意技能点数的合理分配

### 编辑提示
- 拖拽画布可移动技能树视图
- 鼠标滚轮可缩放技能树（0.5x - 2x）
- 悬停技能节点可查看详情Tooltip
- 编辑模式下点击节点可拖拽移动位置

### 编辑模式操作
- **进入编辑模式**：点击顶部"编辑模式"按钮
- **移动节点**：点击并拖拽技能节点到目标位置
- **移动画布**：拖拽空白区域移动视图
- **缩放画布**：鼠标滚轮缩放
- **网格辅助**：编辑模式下自动显示虚拟网格线
- **编辑属性**：使用右侧编辑器面板修改节点属性
- **退出编辑模式**：再次点击"编辑模式"按钮

### Build管理
- 保存的Build会存储在浏览器本地
- 支持重命名和删除Build
- 导出的JSON可用于分享给其他玩家

## 🎯 最佳实践

1. **规划Build**：先确定职业方向（近战/远程/魔法）
2. **核心优先**：先点满核心技能提升基础伤害
3. **平衡攻防**：合理分配防御和攻击技能
4. **终极技能**：根据玩法选择合适的终极技能
5. **保存备份**：定期导出配置防止数据丢失
6. **使用编辑模式**：自定义技能树布局以适应不同需求
