# 伤害计算器 3.0

> Diablo IV Damage Calculator v3.0

精确计算暗黑4角色伤害的工具，支持多职业、多词缀乘区、暴击/易伤/压制机制的完整伤害模拟。

## 🎯 功能特性

### 职业系统
- **7种职业支持**：野蛮人、德鲁伊、法师、死灵法师、游侠、圣骑士、术士
- **职业独立配置**：每个职业有独立的属性换算（力量/敏捷/意志）
- **动态主属性**：根据职业自动切换主属性标签（力量/敏捷/意志）

### 伤害计算系统
- **多乘区计算**：
  - A类词缀区（加法区）
  - B类词缀区（分组乘法区）
  - 独立X区（独立乘法区）
- **暴击机制**：
  - 暴击几率（百分比）
  - 暴击伤害加成（固定+50%基础）
  - 启用/禁用开关
- **易伤机制**：
  - 易伤基础伤害（20%）
  - 额外易伤加成
  - 易伤覆盖率（期望值计算）
- **压制机制**：
  - 压制层数
  - 每层压制加成
- **DOT伤害**：支持持续性伤害计算，自动忽略暴击

### 输入功能
- **主/副武器伤害**：支持双手武器或双持配置
- **攻速**：攻击速度影响DPS计算
- **技能百分比**：技能基础伤害系数
- **主属性**：根据职业自动适配
- **怪物减伤**：支持护甲减伤模拟
- **实时计算**：输入即更新，无需点击计算按钮

### 词缀管理
- **A类词缀**：加法区词缀，支持添加、删除、禁用
- **B类词缀**：分组乘法区，同名叠加后再与其他组相乘
- **独立X词缀**：独立乘法区，各自独立相乘
- **禁用词缀**：支持暂时禁用词缀（不删除数据）

### 结果展示
- **伤害分解**：
  - 武器基础伤害
  - 技能加成后
  - 主属性加成后
  - A类词缀加成后
  - B类词缀加成后
  - 暴击加成后
  - 易伤加成后
  - 独立乘区加成后
- **最终伤害**：普通命中 / 暴击命中
- **DPS计算**：
  - 基础DPS
  - 期望DPS（含暴击和易伤期望）

### 方案管理
- **新建方案**：创建空白方案
- **复制方案**：克隆当前方案
- **切换方案**：在多个方案间切换
- **重命名方案**：修改方案名称
- **删除方案**：删除不需要的方案
- **快照系统**：保存最多5个历史快照
- **导出方案**：导出为JSON文件
- **导入方案**：从JSON文件导入

### 多计算器对比
- **独立计算器实例**：支持同时打开多个计算器进行伤害对比
- **最多4个计算器**：1个主计算器 + 最多3个新增计算器
- **独立状态管理**：每个计算器拥有独立的输入、计算结果和方案
- **标题自定义**：每个计算器可自定义名称
- **快速对比**：直观对比不同属性/词缀配置下的伤害差异
- **保存/加载**：每个计算器支持独立的方案保存和加载
- **导出/导入**：支持JSON格式的方案导出和导入

### 参考BD
- **BD数据展示**：内置参考Build数据
- **一键加载**：快速加载参考Build配置

### UI设计
- **暗黑4主题风格**：
  - 主色调：暗红色(#a50905)、深灰背景(#303030)、金色文字(#d2c8ae)
  - 卡片式布局，分区显示
  - 悬停效果和交互反馈
- **响应式设计**：适配不同屏幕尺寸
- **居中布局**：单个计算器居中显示，多计算器并排从左排列
- **上下结构**：所有计算器统一采用输入面板在上、结果面板在下的布局
