# 巅峰盘模拟器 使用指南

## 🎮 基本操作

### 1. 选择巅峰盘

1. 点击左侧面板的巅峰盘按钮
2. 系统会切换到对应的巅峰盘视图
3. 核心盘始终显示，可选择4个可选巅峰盘

### 2. 解锁节点

1. 查看画布上的可点击节点（绿色虚线圆环标记）
2. 点击可解锁的节点
3. 系统自动高亮显示从入口点到该节点的路径
4. 属性面板实时更新

### 3. 相邻解锁规则

- **入口点**：可以直接解锁（顶部入口、底部链接、巅峰起始节点）
- **普通节点**：必须与已解锁节点**上下左右相邻**才能解锁
- **对角线**：对角线相邻不算有效连接
- **取消解锁**：再次点击已解锁的节点可取消

### 4. 取消节点

1. 再次点击已解锁的节点
2. 节点取消解锁
3. 后续依赖该节点的解锁状态会自动更新

### 5. 装备雕纹

1. 在右侧面板选择雕纹
2. 点击雕纹装备到中心槽位
3. 雕纹效果会影响周边节点

### 6. 导出配置

1. 进入编辑模式
2. 点击"导出配置"按钮
3. JSON配置自动复制到剪贴板

## 🔧 编辑模式

### 1. 进入编辑模式

1. 点击顶部"编辑模式"按钮
2. 画布显示21×21网格辅助线
3. 右侧面板切换为编辑面板

### 2. 节点拖拽

1. 点击并拖拽节点
2. 节点自动吸附32px网格
3. 松开鼠标完成位置更新

### 3. 添加节点

1. 点击"添加节点"按钮
2. 选择节点类型
3. 设置节点属性
4. 点击确认添加

### 4. 删除节点

1. 选中要删除的节点
2. 点击"删除节点"按钮
3. 确认删除操作

## 📊 界面说明

### 主区域（Canvas）
- 21×21网格巅峰盘展示（672px × 672px）
- 节点按类型区分颜色和样式：
  - 普通节点：金色圆形
  - 稀有节点：蓝色圆形+◆
  - 传奇节点：橙色圆形+★
  - 空节点：虚线圆
  - 链接节点：金色圆形+🔗
  - 巅峰起始节点：金色圆形+👑
- 高亮路径显示（金色发光线条）
- 可点击节点提示（绿色虚线圆环）

### 左侧面板
- 巅峰盘选择器（核心盘 + 9个可选盘）
- 巅峰点数显示（总数/已消耗）
- 编辑模式切换按钮

### 右侧面板
- **浏览模式**：属性汇总、节点详情、雕纹列表
- **编辑模式**：节点属性编辑、添加/删除节点、导出配置

### 顶部栏
- 职业选择器
- 缩放控制（0.5x - 2x）
- 重置视图按钮
- 编辑模式切换

## 💡 提示

### 解锁技巧
- 从入口点开始逐步解锁相邻节点
- 优先解锁高价值的稀有/传奇节点
- 注意节点连接路径，避免孤立节点

### 编辑提示
- 拖拽时节点会自动吸附到32px网格
- 编辑模式下显示网格辅助线便于定位
- 导出的配置可用于备份或分享

### 雕纹使用
- 雕纹装备在中心槽位
- 不同雕纹有不同的影响半径
- 雕纹效果需要满足条件才能激活

## 🎯 最佳实践

1. **规划路径**：先规划好从入口到目标节点的路径
2. **优先传奇**：传奇节点提供强大的被动效果
3. **利用雕纹**：合理选择雕纹增强周边节点效果
4. **保存配置**：定期导出配置备份
